# Blackjack Rules Reference

## Card Values
| Card | Value |
|------|-------|
| 2–10 | Face value |
| J, Q, K | 10 |
| A | 11 (or 1 if over 21) |

## Objective
Get a hand value closer to **21** than the dealer without going over (busting).

## Blackjack
A hand of exactly 21 with the first two cards (an Ace + a 10-value card).
- Pays **1.5×** the bet.
- If both player and dealer have blackjack, it is a **tie** (push).

## Actions
| Action | Description |
|--------|-------------|
| **Hit** | Draw another card |
| **Stand** | Keep your current hand |
| **Split** | Split two cards of equal value into two separate hands (max 4 splits) |
| **Double Down** | Double your bet and receive exactly one more card |
| **Surrender** | Forfeit the hand and lose only **50%** of your bet |
| **Insurance** | Side bet (up to half your main bet) when dealer shows Ace; pays 2:1 if dealer has blackjack |

## Dealer Rules
- Dealer must **hit** until reaching a score of **17 or higher**.
- Dealer stands on **17**.

## Winning & Losing
| Result | Outcome |
|--------|---------|
| Player > Dealer (no bust) | Player wins bet |
| Dealer > Player (no bust) | Player loses bet |
| Both bust | Player loses |
| Equal score | Push (tie) — no money exchanged |
| Player busts | Player loses bet |
| Dealer busts | Player wins bet |
