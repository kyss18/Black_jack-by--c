# Blackjack Rules

## Card Values

| Card | Value |
|------|-------|
| 2–10 | Face value |
| J, Q, K | 10 |
| A | 11 (counts as 1 if hand would bust) |

## Objective

Get closer to **21** than the dealer without going over (busting).

## Blackjack

A natural 21 with the first two cards (Ace + any 10-value card).
- Pays **1.5×** the bet
- If both player and dealer have Blackjack → **Push** (tie, no money exchanged)

## Player Actions

| Action | Condition | Effect |
|--------|-----------|--------|
| **Hit** | Any time | Draw one more card |
| **Stand** | Any time | End your turn |
| **Split** | First two cards have equal value | Split into two separate hands; each gets one more card. Max 4 splits per round |
| **Double Down** | First two cards only | Double the bet, draw exactly one more card |
| **Surrender** | First two cards only | Forfeit the hand, lose **50%** of the bet |
| **Insurance** | Dealer's first card is an Ace | Side bet up to half the main bet; pays **2:1** if dealer has Blackjack |

## Dealer Rules

- Dealer always hits until reaching **17 or higher**
- Dealer stands on 17

## Outcomes

| Situation | Result |
|-----------|--------|
| Player > Dealer (no bust) | Player wins bet |
| Dealer > Player (no bust) | Player loses bet |
| Equal scores | Push — no money exchanged |
| Player busts | Player loses bet regardless of dealer |
| Dealer busts | Player wins bet |
| Player Blackjack, dealer no Blackjack | Player wins 1.5× bet |
| Both Blackjack | Push |
| Player surrenders | Player loses 50% of bet |
| Insurance wins | Pays 2:1 on the insurance bet |

## Shoe

- 3 decks shuffled together
- Reshuffles automatically when fewer than 25% of cards remain
